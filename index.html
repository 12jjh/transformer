Transformer 모델의 Sampling 파라미터 분석 보고서

1. 서론

1.1 목적

본 실험의 목적은 Transformer 모델에서 사용되는 temperature, top-k, top-p 파라미터가 출력 단어 선택에 어떤 영향을 미치는지 분석하는 것이다.

특히 각 파라미터가 확률 분포와 결과 생성 방식에 미치는 영향을 비교하고, 모델의 출력 특성이 어떻게 변화하는지를 이해하는 것을 목표로 한다.

1.2 가설

본 실험에서는 다음과 같은 가설을 설정하였다.

1. k는 출력되는 단어의 개수를 결정할 것이다.
2. top-p는 일정 확률 기준에 따라 선택되는 단어 수가 결정되며, 2ⁿ 형태로 증가할 것이다.
3. temperature는 확률을 재분배하여 특정 단어에 집중되거나 분산되게 할 것이다.

2. 실험 결과 해석

2.1 Temperature에 따른 변화

temperature가 낮을 경우 특정 단어의 확률이 매우 높게 나타났으며, 모델이 하나의 단어를 확신하고 선택하는 경향을 보였다.

반면 temperature가 증가할수록 각 단어의 확률이 점점 비슷해졌으며, 여러 단어가 유사한 확률을 가지게 되었다.

이는 temperature가 확률 분포를 평탄하게 만들어 모델의 출력이 더 다양해지도록 만든다는 것을 의미한다.

2.2 k 값에 따른 변화

k 값이 증가할수록 더 많은 단어가 출력 결과에 포함되었으며, 각 단어의 확률이 더 균등하게 분배되는 경향이 나타났다.

이는 k가 작을 경우 상위 단어만 선택되지만, k가 커질수록 하위 확률 단어까지 포함되기 때문이다.

따라서 k는 출력 후보의 개수를 제한하는 역할을 한다.

2.3 top-p에 대한 해석

top-p는 확률이 높은 단어부터 누적하여 총합이 일정 값에 도달할 때까지 선택하는 방식이다.

이로 인해 확률이 한쪽에 몰려 있을 경우 적은 단어만 선택되며, 확률이 분산되어 있을 경우 더 많은 단어가 선택된다.

이는 top-k와 달리 고정된 개수가 아닌, 확률 분포에 따라 선택되는 단어 수가 달라진다는 특징을 가진다.

3. 종합 분석

세 가지 파라미터를 종합하면 다음과 같이 정리할 수 있다.

- temperature → 확률 분포의 형태 조절
- k (top-k) → 출력 단어 개수 제한
- top-p → 확률 기준으로 선택 범위 결정

4. 가설 검증

1. k는 단어 개수를 결정한다 → 성립
2. temperature는 확률을 재분배한다 → 성립
3. top-p가 2ⁿ 형태로 증가한다 → 성립하지 않음

top-p는 실제로 2ⁿ이 아니라 확률 누적 기준으로 동작한다는 점에서 초기 가설과 차이가 있었다.

5. 결론

Transformer 모델의 출력은 확률 분포를 기반으로 하며, temperature, k, top-p는 이 확률 분포를 조절하는 핵심 요소이다.

특히 temperature는 출력의 다양성을 결정하고, k와 top-p는 선택 범위를 제한함으로써 모델의 생성 결과를 제어할 수 있다.
